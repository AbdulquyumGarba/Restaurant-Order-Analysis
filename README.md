# Restaurant Order Analysis
## About the Project
#### This project will look into Order details such as (*Average Price of Items Per Category, Lowest Price and Highest Price Per Category, Total Items Per Category, Frequency of Order by Time, Frequency of Order by Day, What Percentage of Order was below Average, What Percentage of Order was Above Average*)
